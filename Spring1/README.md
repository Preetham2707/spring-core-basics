# Spring Core Basics

This project demonstrates the fundamental concepts of the Spring Framework.


## Concepts Covered

* Bean
* IoC (Inversion of Control)
- Dependency Injection
  - Setter Injection
  - Constructor Injection
- Bean-to-Bean Injection
- Reference Injection
- BeanFactory
- ApplicationContext
- Spring Container
- XML Configuration
- Collection Injection
  - List Injection
  - Set Injection
  - Map Injection

## Technologies Used

* Java
* Spring Framework
* Eclipse IDE

## Project Structure

* Student.java – Bean class
* Test.java – Main class to retrieve bean from Spring Container
* SpringConfig.xml – Spring XML configuration
